Room Dead-Minutes Analysis

Every section occupies an 80-minute booking window regardless of actual class length. Dead minutes = unused tail of that window + any idle time above the required 20-min turnaround before the next booking in the same room. Lower is better. Click a room to see its day-by-day breakdown.

HC-7 · HARD CONSTRAINT
Uniform block across all meeting days. A section's time block is identical on every day it meets (MWF or TR all start at the same assigned block).
SC-9 · SOFT CONSTRAINT
Room dead-minutes are penalized, but not enforced as a hard constraint. Gaps > 20 min between bookings in the same room are discouraged, but allowed if needed for feasibility.
