Room Dead-Minutes Analysis

Dead minutes are computed per active room/day by scanning each universal 80-minute block and counting minutes the room is not used. This includes unused minutes inside occupied blocks plus fully unused blocks before, between, and after classes. Lower is better.

HC-7 · HARD CONSTRAINT
Uniform block across all meeting days. A section's time block is identical on every day it meets (MWF or TR all start at the same assigned block).
SC-9 · SOFT CONSTRAINT
Room dead-minutes are penalized, but not enforced as a hard constraint. The optimizer prefers compact room/day usage with fewer unused 80-minute blocks.
